## Background: how calcium signalling works in platelets

Platelet activation produces a rapid rise in cytosolic calcium (Ca²⁺), which
triggers shape change, granule secretion, and aggregation. The calcium signal
is generated by a tightly coupled set of pumps, channels, and sensors that
span three compartments: the cytosol, the Dense Tubular System (DTS, an
intracellular calcium store), and the extracellular space.

![Calcium signalling flow diagram](/Users/steve/github/wcEcoli/reports/figures/calcium-signal-flow.png)

In summary: ADP binds P2Y1 → Gq activates PLCβ → PLCβ cleaves PIP2 to release
IP3 → IP3 opens the IP3R channel in the DTS membrane, flooding Ca²⁺ into the
cytosol → the Ca²⁺ rise drives activation. Three processes then restore resting
state: SERCA pumps Ca²⁺ back into the DTS; PMCA ejects Ca²⁺ to the outside;
and when the DTS empties, STIM1 senses this and gates Orai1 channels (SOCE) so
that extracellular Ca²⁺ can restock the DTS.

## Key term definitions

| Term | Stands for | Role in model |
|------|-----------|---------------|
| **Ca²⁺** | Calcium ion | The signalling molecule whose cytosolic concentration drives platelet activation |
| **DTS** | Dense Tubular System | Intracellular calcium store unique to platelets; equivalent to the endoplasmic/sarcoplasmic reticulum in other cells |
| **IP3** | Inositol 1,4,5-trisphosphate | Cytosolic second messenger produced by PLCβ; binds to IP3R to open the DTS store |
| **IP3R** | IP3 Receptor | Ca²⁺-permeable channel embedded in the DTS membrane; opens when IP3 binds; has complex 6-state gating including Ca²⁺-dependent inactivation |
| **SERCA** | Sarco-Endoplasmic Reticulum Ca²⁺-ATPase | ATP-powered pump that moves Ca²⁺ from cytosol back into the DTS; acts as the "refill" mechanism |
| **PMCA** | Plasma Membrane Ca²⁺-ATPase | ATP-powered pump on the outer membrane that ejects Ca²⁺ from the cell entirely; activated by calmodulin |
| **STIM1** | Stromal Interaction Molecule 1 | Ca²⁺ sensor protein sitting in the DTS membrane; when DTS Ca²⁺ falls, STIM1 clusters and moves to trigger Orai1 |
| **Orai1** | — (also called CRACM1) | Pore-forming subunit of the CRAC channel (Calcium Release-Activated Calcium channel) in the plasma membrane; gated by STIM1 |
| **SOCE** | Store-Operated Ca²⁺ Entry | The overall process by which DTS depletion (sensed by STIM1) opens Orai1 channels and lets extracellular Ca²⁺ refill the cell |
| **CRAC** | Calcium Release-Activated Calcium | The ion channel formed by Orai1; opened by STIM1 binding; carries the SOCE current |
| **PLC** | Phospholipase C | Enzyme that cleaves PIP2. PLCβ isoforms are activated by GPCRs (ADP → P2Y1 → Gq → PLCβ). PLCγ isoforms by receptor tyrosine kinases (GPVI → Syk → PLCγ2) |
| **PIP2** | Phosphatidylinositol 4,5-bisphosphate | Membrane phospholipid substrate for PLCβ; hydrolysed to yield IP3 (cytosolic) + DAG (membrane) |
| **DAG** | Diacylglycerol | Membrane lipid product of PIP2 hydrolysis; activates PKC; not directly modelled in calcium core |
| **P2Y1** | Purinergic receptor Y1 | G-protein coupled receptor for ADP on platelet surface; activation leads to Gq→PLCβ→IP3 cascade |
| **Gq / PLCβ** | Heterotrimeric G protein Gq; Phospholipase Cβ | Signal transduction bridge between P2Y1 and IP3 production |
| **CaM** | Calmodulin | Ubiquitous Ca²⁺-binding protein; activates PMCA when Ca²⁺ is high; ~20,000 copies/platelet |
| **MCU** | Mitochondrial Calcium Uniporter | Channel that takes up Ca²⁺ into mitochondria when cytosolic Ca²⁺ is high; acts as buffer and integrator |
| **PKG / PKA** | Protein Kinase G / Protein Kinase A | Kinases activated by cGMP / cAMP; phosphorylate IP3R and other targets to inhibit activation (inhibitory pathway) |

## IP3 forcing function — Dolan 2014 (Supporting Material Fig S2)

Dolan 2014 uses a piecewise approach:
1. **Interpolation fit** — cubic Hermite interpolant of measured human platelet [IP3]
   time-course data (ADP stimulation). Data only to 30s; extrapolated beyond.
2. **Parametric form (Eq S1)** — for dose-response simulations:

```
[IP3](t) = IP3_rest × f(t)
```

Where f(t) is an exponential function with parameters a, b, c and exponent 0.6.
- Second rational function term allows [IP3] to settle above baseline at long times.
- Shape: rises ~5-fold within 5 s, returns to new steady level within 200 s.

**For our Milestone 1 implementation**, we can use a simplified version:
```python
def ip3_forcing(t, ip3_rest, fold=5.5, t_peak=3.0, tau_decay=60.0):
    """IP3 forcing function (Dolan 2014 Fig S2 shape)."""
    if t <= 0:
        return ip3_rest
    rise = fold * (1 - math.exp(-t / t_peak))
    decay = math.exp(-(t - t_peak) / tau_decay) if t > t_peak else 1.0
    return ip3_rest * (1.0 + (fold - 1.0) * rise * decay)
```

This captures the essential behaviour: fast rise, slow return to slightly above baseline.

### SERCA cycle (type 3b isoform kinetics; ref: Dode 2002, as cited via Purvis 2008 Table 1)

| Step | Forward | Reverse |
|------|---------|---------|
| E2 ⇌ E1 (shuttling) | k₁ = 600 s⁻¹ | k₋₁ = 600 s⁻¹ |
| E1 + 2Ca²⁺cyt ⇌ E1·Ca²⁺₂ | k₁ = 1×10¹⁵ M⁻²·s⁻¹ | k₋₁ = 10 s⁻¹ |
| E1·Ca²⁺ → E1P·Ca²⁺ (phosphorylation) | k₁ = 700 s⁻¹ | k₋₁ = 5 s⁻¹ |
| E1P·Ca²⁺ ⇌ E2P·Ca²⁺ (conformational) | k₁ = 600 s⁻¹ | k₋₁ = 50 s⁻¹ |
| E2P·Ca²⁺ → E2P + 2 Ca²⁺_dts (transport) | k₁ = 1000 s⁻¹ | k₋₁ = 4×10⁹ M⁻²·s⁻¹ |
| E2P → E2 (dephosphorylation) | k₁ = 500 s⁻¹ | k₋₁ = 1 s⁻¹ |

**Verified ✓ (Zotero, Purvis 2008 itemKey Z5W8RMLQ, 2026-04-23)** — all six rate
constants match Purvis Table 1 SERCA cycle. Stoichiometry of the transport step
corrected from "+ Ca²⁺" to "+ 2 Ca²⁺_dts" (the M⁻² units of k₋₁ confirm that
two Ca²⁺ ions are released into the DTS lumen on each turnover, consistent with
the 2:1 Ca²⁺:ATP stoichiometry of SERCA3b).

**Verified ✓ (Sneyd & Dufour 2002 PDF audit, 2026-05-08)** — all rate
constants match the original Sneyd & Dufour 2002 *PNAS* 99:2398–2403
Fig 4 caption, with two corrections to the previous Purvis-2008-derived
revision of this table:

1. **k₃ corrected: 11 → 0.11 s⁻¹** (factor 100×). Sneyd-Dufour 2002
   Fig 4 caption gives `k₃ = 0.11 s⁻¹·µM⁻¹` (the µM⁻¹ in the unit is
   a typo — the φ₃ formula `k₃·L₅/(L₅+c)` is dimensionally consistent
   only when k₃ is s⁻¹; their body text confirms "φ₃ ≈ 0.1 s⁻¹"). The
   Purvis 2008 Table 1 figure of "11 s⁻¹" appears to be a 100×
   transcription error from the primary source. This affects the
   o → s "shutting" rate of the Sneyd-Dufour ladder; correcting it
   collapses the s state population from ~3.8% to ~0.04% at the
   IP3 = 50 nM, ca_cyt = 100 nM resting equilibrium, matching Dolan
   Table S1's reported `s ≈ 0`.

2. **k₋₄ unit corrected: µM⁻¹·s⁻¹ → s⁻¹**. Sneyd-Dufour Fig 4 caption
   gives `k₋₄ = 0.54 s⁻¹` and the φ₋₄ formula `L₁(k₋₄+l₋₆)/(L₁+c)`
   requires k₋₄ in s⁻¹ for dimensional consistency. The previous
   `µM⁻¹·s⁻¹` annotation was a documentation typo; the value (0.54)
   was always right.

Earlier (2026-04-23) revisions of this table also omitted **L₃ = 0.025 µM**
from the n→i1 row; this is a real Sneyd/Purvis equilibrium constant for the
inhibitory site and is required by the rate-law expression.
Add it back when implementing the rate laws (issue #25).

### MWC implementation (closes issues #45/#46)

The code in `reconstruction/platelet/dataclasses/process/calcium_signalling.py`
now implements the full Hoover & Lewis 2011 / Dolan 2014 MWC scheme. The
SOCE block has three layers:

1. **STIM1 mass-action cycle** (DTS-side):
   ```
   STIM1·Ca²⁺_dts  ⇌  STIM1_free + Ca²⁺_dts    (EF-hand release)
   2 STIM1_free   ⇌  STIM1_dim                 (dimerisation)
   ```
   Rate constants `k_release_r = 3.475×10⁻³ µM⁻¹s⁻¹` and
   `k_dim_f = 1.15×10⁻⁴ count⁻¹s⁻¹` are derived from detailed balance at the
   Dolan Table S1 IC — this gives `v = 0` for both reactions at rest.

2. **Puncta entry** (Dolan 2014 Eq. 2):
   ```
   (STIM2)p = qp · STIM2_dimers
   qp = α · ([Ca²⁺]_cyt^n / (KM^n + [Ca²⁺]_cyt^n)) + 0.01
   ```
   with `α = 0.2` (Dolan default), `KM = 0.5 µM`, `n = 4` (mid-range from
   Dolan's scanned ranges; not yet refit to platelet homeostatic constraints).

3. **MWC channel gating** (Hoover Fig. 4B):
   ```
   For i = 0..4 STIM dimers bound to a tetrameric Orai channel:
     [CSi]/[C]   = (4 choose i) · a^(i(i−1)/2) · (Ka·Sf)^i
     [OSi]/[CSi] = f^i · L
   ```
   with Hoover values `L = 10⁻⁴, f = 14.2, a = 0.5` (binding cooperativity).
   `Ka = 2` is rescaled from Hoover's a.u. (where `Ka_Hoover = 100`,
   `Stotal_Hoover = 3.2 a.u.`) to match the platelet saturating dimer count
   (~170 dimers) at the same `Ka·Sf ≈ 300` saturation point.

The SOC current uses Dolan Eq. 4 with single-channel conductance
`γ_SOC = 0.3 fS`, an effective value derived from the resting balance
constraint `SOCE_rest = PMCA_steady_rest`. Literature γ_SOC for CRAC is
~24 fS at saturating Po, but for the integer-count regime at platelet rest
(<1 channel open) the effective conductance is reduced to avoid spurious
runaway. See `reports/platelet-calcium-calibration.md` §3 and §6 for the
full derivation.

## Molecule count conversions

For 6 fL cytosol:
- 100 nM = 100×10⁻⁹ × 6.022×10²³ × 6×10⁻¹⁵ = **~361 molecules**
- 50 nM = **~181 molecules** (IP3 resting)

For 0.12 fL DTS (2% of cytosol):
- 250 µM = 250×10⁻⁶ × 6.022×10²³ × 0.12×10⁻¹⁵ = **~18,066 molecules** Ca²⁺ in store

For 0.25 fL DTS (4% of cytosol):
- 250 µM = **~37,638 molecules** Ca²⁺ in store

**These numbers matter** — at 100 nM cytosolic Ca²⁺ there are only ~360 ions.
The integer count representation in the engine is at the edge of continuum validity.
Stochastic effects at this level are biologically real (Purvis 2008, Sveshnikova 2015
both use stochastic simulation for this reason).

## Sveshnikova 2025 model (confirmed from online HTML)

The online full text confirms the model from ref [37] (Sveshnikova 2015):
- **29 species**, 27 differential equations, 2 fixed (Caex, thrombin)
- **4 compartments:** cytosol (3 fL), DTS (1.5 fL), mitochondria (0.3 fL), PM equivalent
- **Key insight from review:** PLC-Gq complex never exceeds ~1 molecule/cell at any time.
  This is the stochastic bottleneck — all downstream variability traces back here.
- Calcium oscillation frequency: 0.3–0.4 Hz (at medium stimulation)
- Signal shape transformation: step (thrombin) → peak (PAR1-Gq-GDP) → oscillation
  (cytosolic Ca²⁺) → peak (mitochondrial Ca²⁺) → threshold (mPTP opening)
- Mitochondria act as an **integrator** of the oscillatory calcium signal
- mPTP opening occurs when time-integral of [Ca²⁺]cyt exceeds a threshold

**Note:** The Sveshnikova 2025 Table 2 contains all reaction equations and parameters
but they are embedded as images in the HTML and not machine-readable. The parameter
values are referenced as coming from refs [37] (Sveshnikova 2015), [49], [50], [51],
[52], [53]. The 2015 paper used COPASI software with parameters tuned to match
experimental calcium oscillation data.

## Discrepancies and open questions

1. **DTS volume:** Purvis/Dolan use 1–5% of cytosol (0.06–0.3 fL). Sveshnikova 2025
   uses 1.5 fL (50% of their 3 fL cytosol). This is a 5–25× difference. May explain
   why Sveshnikova model sustains larger oscillations. **Decision needed.**

2. **Cytosol volume:** Purvis says 6 fL; Sveshnikova 2025 says 3 fL. Likely reflects
   different definitions (total cell vs cytosol only vs non-granular volume).
   Purvis references the measurement explicitly.

3. **SERCA copy number:** Burkhart gives ~25,000 (combined isoforms). Purvis model
   estimates ~10⁶ for high-response configurations. Dolan 2014 Table S1 gives a
   **representative filtered value of 11,892** — consistent with Burkhart proteomics.
   The Purvis 10⁶ value was from "high-response" configurations that likely
   represent unrealistic extremes.

4. **Resting [Ca²⁺]dts:** Range from 250 µM (Dolan, measured) to 1000 µM
   (Sveshnikova 2025). This significantly affects store release magnitude.
   Dolan's value is a direct Fluo-5N measurement; Sveshnikova's is a model fit.

5. **VIM (DTS membrane potential):** Unmeasured in platelets. Dolan 2014 identifies
   this as the most uncertain parameter and shows it must be > −70 mV for functional
   SOCE. Clustering analysis (Fig S4) confirms responsive configurations have
   **high VIM** (near −60 mV). Critical for Nernst driving force.

6. **SOCE parameters (L, Ka, n, KM):** Dolan 2014 scans these as unknowns. Their
   clustering analysis shows the model is **insensitive to n and KM** — SOCE flux is
   dominated by free STIM2 availability (regulated by Ca²⁺dts), not puncta formation.
   This simplifies our implementation.

7. **IP3R gating kinetics:** Both Purvis and Sveshnikova use Sneyd & Dufour 2002 type 2
   model (6 states). The Dolan Table S1 shows representative state fractions at rest.
   **Decision: implement the full 6-state model** — the parameters are well-characterised
   and oscillatory behaviour requires it.

8. **IP3R copy number:** Burkhart gives ~4,850 total (all isoforms). Dolan Table S1
   representative gives 1,328. This is within the Purvis "mild-response" range
   (1,000–5,000). The discrepancy may reflect that not all proteomic copies are
   functionally active in DTS membrane.

9. **Stochastic bottleneck:** Sveshnikova 2025 identifies PLC-Gq complex (~1 molecule)
   as the source of all downstream stochasticity. This is below the continuum limit
   and means our deterministic ODE approach will miss cell-to-cell variability in the
   upstream module. Acceptable for Milestone 1 (deterministic), flag for Milestone 3.

## Phasing recommendation

### Milestone 1 (calcium core): species needed

Minimum set to demonstrate a Ca²⁺ transient:

```
CA2_CYT[c]       — cytosolic free Ca²⁺
CA2_DTS[dts]     — DTS stored Ca²⁺
IP3[c]           — inositol trisphosphate (IP3 as forcing function initially)
IP3R[dts]        — IP3 receptor (possibly multiple states: n, o, a, i1, i2, s)
SERCA[dts]       — SERCA pump (possibly multiple cycle states)
PMCA[m]          — plasma membrane Ca²⁺-ATPase
```

Optional for Milestone 1 (add if needed for stable dynamics):
```
STIM1[dts]       — Ca²⁺ sensor
ORAI1[m]         — SOC pore
CA2_EX[e]        — extracellular Ca²⁺
```

### Implementation choice: IP3R complexity

| Option | States | Pro | Con |
|--------|--------|-----|-----|
| Full Sneyd model | 6 (n, o, a, i1, i2, s) | Matches Purvis/Dolan exactly | Complex, many parameters |
| Simplified Hill | 1 (Po as function of [IP3], [Ca²⁺]) | Easier to test and explain | May not reproduce oscillations |

**Recommendation:** Start with the full 6-state IP3R. The parameters are well
documented, and oscillatory behaviour (which is biologically important) emerges
from the biphasic Ca²⁺ dependence of IP3R — this cannot be captured by a simple
Hill function.
