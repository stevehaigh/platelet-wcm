# Archive — historical / superseded reports

Design, data, and planning documents from the early model (≈ v0.2–v0.5,
2026-04 to 2026-05) plus a corrupted thesis-review scratch file, moved here on
2026-06-24 during the submission spring-clean. **None of this is current.**

They are kept for provenance — the development trail behind the model — not as a
description of how the model works now. For the current state, see:

- `reports/design/` — current design docs (start with its `README.md`, then
  `validation-map-2026-06-19.qmd` for the model-status overview)
- `reports/experiments/` — the current demo-experiment write-ups
- `CLAUDE.md` — the authoritative current architecture

Version labels inside these files (v0.2 / v0.3 / v0.4.1, "Phase 1/2/3",
"21 species", forced-IP₃, in-place monkeypatch knobs, etc.) describe the model
*at the time of writing* and are expected to be out of date.
