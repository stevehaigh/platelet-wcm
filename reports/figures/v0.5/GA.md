# Figure legend — `GA.png` (graphical abstract)

**Graphical abstract — building a whole-cell model of the platelet.**
Relevant literature is curated and verified before its values are used to
generate running model code; AI and code tools assist with this process. The
code runs on a laptop and produces output (such as cytosolic Ca²⁺ levels)
from inputs (such as agonist concentrations). The output is then plotted
against reference data to verify model performance.

Single-panel summary of the project's contribution and workflow — intended to
sell *"the whole-cell approach is worth pursuing"* rather than report a
specific result. Keep total text under ~30 words at final render; ≥300 dpi.
